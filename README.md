# FlexiCalc - Webes Számológép Projekt

Egy funkcionális, rácsos elrendezésű (CSS Grid) számológép, amely tiszta JavaScript logikára épül.

## Technikai részletek
* **Matematikai logika:** String formátumú matematikai kifejezések kiértékelése és lefutása JavaScript környezetben.
* **Kivételkezelés (Try-Catch):** A hibás szintaxisok (pl. két egymás utáni műveleti jel) intelligens lekezelése, megakadályozva az alkalmazás összeomlását.
* **Modern CSS Grid:** Reszponzív, gombokból álló felhasználói felület tervezése.
